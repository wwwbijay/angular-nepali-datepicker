Angular Nepali Date Picker

A simple Nepali Date Picker (Bikram Sambat) library in angular with no dependancy.

  • Examples
    Basic Usage

    This is basic usage.

    Output:

    day: {{ nepaliDate1.day }},

    month: {{ nepaliDate1.month }},

    year: {{ nepaliDate1.year }}

    Code

    <np-datepicker [(ngModel)]="nepaliDate"> </np-datepicker>
    Language

    We can choose to select language to nepali 'ne' or english 'en'.

    Output:

    day: {{ nepaliDate2.day }},

    month: {{ nepaliDate2.month }},

    year: {{ nepaliDate2.year }}

    Code

    <np-datepicker [(ngModel)]="nepaliDate" language="en" > </np-datepicker>
    Theme

    We can choose to select theme to light 'light' or dark 'dark', or light blue 'lightblue' as well.

    Output:

    day: {{ nepaliDate3.day }},

    month: {{ nepaliDate3.month }},

    year: {{ nepaliDate3.year }}

    Code

    <np-datepicker [(ngModel)]="nepaliDate" theme="light" > </np-datepicker>
