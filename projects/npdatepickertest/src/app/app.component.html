Angular Nepali Date Picker

A simple Nepali Date Picker (Bikram Sambat) library in angular with no dependancy.

  • Examples
    Basic Usage

    This is basic usage.

    Output:

    day: {{ nepaliDate1.day }},

    month: {{ nepaliDate1.month }},

    year: {{ nepaliDate1.year }}

    Code

    <np-datepicker [(ngModel)]="nepaliDate"> </np-datepicker>
